Maestro Extension - Macro Builder

To make a Macro:

  1. Go to the Control Page Page on the Stages tab.
  2. Set all the toggles the way you want them, for each fixture you want to include in the Macro.
  3. Come back to this page and select the fixtures you want to include in your Macro.
  4. Click "Add Macro" to create it.

To use the Macro:

  1. Use it to set the toggles to the stored state.
  2. Clear the Macro to revert all toggles to the previous state.

Remember to back up your Stage and Fixtures before testing this feature for security.
