Connecting to the MaestroDMX
The plugin will work when your MaestroDMX is running and available on the network.
It can work with http://maestro.local or http://10.0.0.200 or http://192.168.37.1 which are the deafult address and ips of the MaestroDMX. By deafult it's looking for maestro.local which should work in most cases

If you are using a different IP, you may need to modify your hosts file to point to the correct IP.
Instructions on how to modify the hosts file can be found, for example at How to Edit Hosts File
where if your maestro was working for example at ip address 192.168.100.100, you would add a line to hosts file like: maestro.local 192.168.100.100
Creating Macros
  1. For your convenience, before you start modifying, take a backup of your Stage, from the MaestroDMX control panel, from this extension, you can take a backup of all the fixture settings and macros via the Backup & Restore tab.
  2. You can set toggles via thethe Control Page Page on the Stages tab. You can also set Pan/Tilt positions via the Pan/Tilt Finder, on the Fixtures Tab below.
    If your fixture(s) have Gobo or Prism, you can disable or enable all of them using the Fixture Toggles tab below.
  3. Set all the toggles the way you want them, for each fixture you want to include in the Macro.
  4. Come back to this page and select the fixtures on the Fixtures tab below you want to include in your Macro.
  5. Click "Add Macro" to create it.
  6. Once you have created the macros you can at any time click Restore below to restore all the fixtures back to their normal state before you began.

To use the Macro:

  1. Use it to set the toggles to the stored state.
  2. Clear the Macro to revert all toggles to the previous state.
  3. Macros are specific to the currently active Stage.
  4. Remember - the Macro applies ALL modified current settings, so if you have modified the Control page, the Pan/Tilt or used a Fixture Toggle below it will be recorded to the macro!

Remember to back up your Stage and Fixtures before testing this feature for your own security.

Enable Macro Triggering
You can trigger a macros activation or deactivation, based on the Activity Level of the music.

  1. Set the Activity Level Threshold on the Macros Tab for the macro you want to be triggered ON.
  2. Set the Activity Level Threshold on the Macros Tab for the macro you want to be triggered OFF.
  3. If you configure ON/OFF parameters for multiple Macros, which affect the same fixtures, then the first macro will win. (Higher in the list of macros will take precedence)
  4. Adjust the Minimum Interval, to prevent the macro from being triggered too often.
  5. Adjust the Minimum Run Time, to prevent the macro from being stopped too quickly.
  6. Unlike other triggers, the Macro trigger will only work if this settings page is open!

Enable Strobe Triggering
If the MaestroDMX Strobe function doesn't work, and your fixture has a dedicated Strobe or Shutter channel, you can set specific parameters in the Fixtures tab below to to fix it.
  1. Set "Shutter Open" value, representing the normal open state for the Shutter or Strobe
  2. Set "Strobe" value, corresponding to the strobe function of the fixture. You can fine-tune the Strobe value according to the DMX values specified in the fixture's manual, enabling adjustment for strobe speed.
  3. If your fixture has multiple Strobe or Shutter channels, you can set some or all of them in the Fixtures tab (For example, the Gigbar Move ILS has more than 10!).
    Refer to your fixtures DMX Table in the manual to determine the correct Open and Strobe values for each channel.
Once the parameters are set correctly, when you use the "Strobe" button on the footer, or in the Show Page it should cause your fixture to strobe. If it doesn't work, try adjusting the values or refreshing the main MaestroDMX page to ensure you have the latest config.
Backup & Restore

Backup Now, will create a Snapshot of the currently loaded stage, all settings in this page and store it locally.
You can download the Config which *includes* the Fixture Backup and all settings in this page.

If you want backup of the settings, without the fixtures, you can download the Config file only via the Download Config button.

Restore Backup can be used to upload the earlier downloaded Backup or Config file.
If you restore a config which contains a Fixture Backup, it will overwrite the current stage's fixture settings!

If you're not sure, go to Stages and Download Stage from the Maestro which will give you a backup of the current stage before you import a config and overwrite the fixtures.

Please note. You can import a file from a different stage, but it will fail if the fixtures do not match.

Autofog can be also switched on/off from the Icon Menu in the Chrome toolbar.

Activity trigger depends on the Audio Activity Level hitting the minimum peak you specify.
The Duration is how long the Fog button will be pressed.
The Interval is the minimum time between each execution of the Autofog.
Some music may have a lot of activity and it would trigger too many fog blasts, this interval helps you avoid this.

You can activate both Fog on Timer, and Fog on Activity Peak, but only one will be executed at a time and they will both respect each others interval settings.

These switches are live edits, and will take effect immediately

Toggle Fixture Settings

The MaestroDMX Interface requires you to switch all toggles one by one, these fixture toggles help to automate the process.
If you label a Gobo or Prism as any of [Open, No Gobo/No Prism, Gobo Off/Prism Off, Gobo Open/Prism Open, None] the toggle will disable all Gobos/Prisms except those.

Please note: If these toggles are active when you create a Macro, they will be saved to the Macro.
These switches are live edits, and will take effect immediately
